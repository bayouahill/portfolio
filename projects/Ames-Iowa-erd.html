Statistical Analysis

Ames Iowa Housing Analysis

Comprehensive statistical analysis exploring housing price determinants in Ames, Iowa through regression modeling, ANOVA testing, and correlation analysis.

Role Data Analyst
Year 2025
Technologies Python, pandas, scipy, seaborn, matplotlib
View Colab

Project Overview

This analysis investigates key factors influencing residential property values in Ames, Iowa using the comprehensive AmesHousing dataset. Through eight targeted research questions, the project employs statistical modeling and visualization techniques to uncover relationships between home characteristics and sale prices. The analysis provides data-driven insights into real estate valuation patterns across different neighborhoods, building types, and quality metrics.

Research Questions Analyzed

  • Relationship between home size (gross living area) and sale price using linear regression
  • Regional price variations across five defined Ames neighborhood zones
  • Association between exterior material quality and property values for single-family homes
  • Impact of construction year on sale price trends
  • Garage type influence on home valuations using ANOVA and Tukey HSD tests
  • Price differential between finished and unfinished basements via t-test analysis
  • Overall quality rating correlation with sale price (R-squared analysis)
  • Chi-squared analysis of building type distribution across neighborhood regions

Key Findings

The analysis revealed a strong positive correlation between home size and sale price, with overall quality rating showing the strongest relationship—each 1-point quality increase correlating with approximately $45,251 in additional value. Geographic analysis demonstrated significant regional pricing disparities, with Northeast Ames averaging $137,721 more than Central Ames properties. Homes with built-in or attached garages, finished basements, and superior exterior materials commanded premium prices. Construction year showed a modest upward trend of roughly $1,475 per year. Statistical testing (ANOVA, t-tests, chi-squared) confirmed significant associations across all examined variables.

Technical Approach

Analysis conducted in Python using pandas for data manipulation and scipy.stats for statistical testing including linear regression, ANOVA, Tukey HSD post-hoc tests, independent t-tests, and chi-squared contingency analysis. Data visualization implemented through seaborn and matplotlib, featuring scatter plots with regression lines, bar charts, and heatmaps. Neighborhood categorization applied domain knowledge to group 28 neighborhoods into five regional zones. Missing values handled appropriately for categorical variables (garage type, basement finish).
